# Presence by Calibre Studio

Static Next.js landing for Presence: the Be Seen // Be Found offering.

- Be Seen (Directed): brand, content and film. See the work at calibrestudio.co/works
- Be Found (Indexed): AI visibility audit at indexed.calibrestudio.co

A duplicate of the Indexed shell (same nav, motion, logo wall and footer) with the
audit form replaced by the two-door layout. No API routes and no env vars: pure
static, deploys free on Vercel at presence.calibrestudio.co.

## Stack
Next.js 14 (App Router, JS), Inter + Fragment Mono, deployed on Vercel.

## Develop
    npm install
    npm run dev      # localhost:3000
    npm run build    # production build
